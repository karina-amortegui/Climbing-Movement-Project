# CLIMBING MOVEMENT PROJECT 

CRUD system to save climbing movements then match movement to exercises. It will have a form for POSTing to Server to save into DB and then relate exercises to movement. Can also manage data in DB with CRUD operations.

## tech stack

```
node.js
express
mongo
docker
```

## start project

```
git clone <repo>
cd <repo>
npm install

- add .env file
```

## start dev mode

```
npm run dev
```

## start prod mode

```
npm run start
```

## stop server

```
ctrl + c
```
